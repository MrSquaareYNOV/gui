# gui

## Using

### Server

Run in development:

```shell script
nx run server:serve
```

Build:

```shell script
nx run server:build
```

### Mobile (User app)

Run in development:

```shell script
nx run mobile:serve
```

Build:

```shell script
nx run mobile:build
```

#### Android

Launch in emulator:

```shell script
nx run mobile:run:android
```

#### iOS

Launch in emulator:

```shell script
nx run mobile:run:ios
```

### Web (Admin app)

Run in development:

```shell script
nx run web:serve
```

Build:

```shell script
nx run web:build
```

## Documentation

### OpenAPI

Generate:

```shell script
npm run openapi:generate
```

Serve:

```shell script
npm run openapi:serve
```
